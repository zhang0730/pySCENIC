### 1. 数据准备（在 R 中进行）

#### 1.1 数据子集选择与表达矩阵导出

- •**选择特定细胞类型**：从 Seurat 对象 `scRNA`中选择特定的细胞类型（例如 `"Plasma"`）。
- •**导出表达矩阵**：将选择的细胞类型的表达矩阵（以转置形式）导出为 CSV 文件 `sce_exp.csv`。
- •**导出细胞元数据**：导出相关的细胞元数据（如细胞类型、RNA 计数和特征数）为 CSV 文件 `cellInfo.csv`。

#### 1.2 转换为 Linux 分析

- •**说明**：在 R 中完成数据准备后，将数据文件转移到 Linux 环境中进行后续的 pyscenic 分析。





### 2. Linux 环境下的 pyscenic 分析

#### 2.1 环境搭建与依赖安装

- •**创建 Conda 环境**：创建一个名为 `pyscenic`的 Conda 环境，并激活该环境。
- •**安装依赖包**：安装必要的 Python 包，如 `numpy`, `cytoolz`, `scanpy`，以及 `pyscenic`。

#### 2.2 数据文件准备

- •**下载必要的文件**：•**转录因子注释文件**：下载小鼠和人类的 motif 到 TF 的映射文件。•**转录因子列表**：下载人类和小鼠的转录因子列表。•**参考数据库**：下载适用于小鼠和人类的基因与 motif 排名的 feather 格式数据库文件。

#### 2.3 文件转换

- •**转换为 Loom 文件**：使用 Python 脚本 `trans.py`将 R 中导出的表达矩阵 `sce_exp.csv`转换为 Loom 文件 `sce.loom`，这是 pyscenic 分析所需的输入格式。

#### 2.4 分析步骤

1. 1.**GRN 构建**：•**目的**：使用 GRNBoost2 算法推断转录因子与基因的共表达模块。•**命令**：`pyscenic grn`，生成 `sce.adj.csv`文件。
2. 2.**TF-motif 富集分析 (RcisTarget)**：•**目的**：进行 TF-motif 富集分析，识别转录因子的直接靶标，生成 regulons（TF 及其调控的靶基因）。•**命令**：`pyscenic ctx`，生成 `sce.regulons.csv`文件。
3. 3.**AUCell 分析**：•**目的**：使用 AUCell 对每个细胞的每个 regulon 活性进行评分。•**命令**：`pyscenic aucell`，生成 `sce_SCENIC.loom`文件，即最终的分析结果。





### 3. 在 R 中进行结果可视化

#### 3.1 加载必要的 R 包

- •**分析包**：如 `SCopeLoomR`, `AUCell`, `SCENIC`。
- •**可视化包**：如 `dplyr`, `ggplot2`, `ComplexHeatmap`, `plotly`等。

#### 3.2 读取 Loom 文件并提取数据

- •**读取 Loom 文件**：使用 `open_loom`函数读取 `sce_SCENIC.loom`文件。
- •**提取 Regulons 和 AUC 数据**：获取 regulons 和 regulon AUC 数据，用于后续分析。

#### 3.3 可视化分析

##### 3.3.1 RSS 分析（Regulon Specificity Scores）

- •**目的**：查看细胞类型特异性转录因子。
- •**步骤**：1.**加载 Seurat 对象并提取元数据**：选择特定细胞类型（如 `"Plasma"`），提取相关的元数据信息。2.**计算细胞特异性 TF**：使用 `calcRSS`函数计算 regulon specificity scores。3.**绘制 RSS 图**：使用 `plotRSS`函数绘制点图，展示不同细胞类型中转录因子的特异性。

##### 3.3.2 热图可视化

- •**目的**：通过热图展示 regulon specificity scores 的分布。
- •**步骤**：1.**数据转换**：将 RSS 数据转换为适合热图绘制的格式。2.**安装和加载额外的 R 包**：如 `ggheatmap`。3.**绘制热图**：使用 `ggheatmap`函数绘制热图，展示不同细胞类型中 regulon 的相对活性，使用自定义颜色和注释。





### 4. 总结

该文档提供了一个完整的流程，从在 R 中准备单细胞表达矩阵和元数据，到在 Linux 环境下使用 pyscenic 进行转录因子分析，最后回到 R 中进行结果的可视化。关键步骤包括：

1. 1.**数据准备**：在 R 中选择特定细胞类型，导出表达矩阵和元数据。
2. 2.**环境配置与数据转换**：在 Linux 中安装必要的工具和依赖，将表达矩阵转换为 Loom 文件。
3. 3.**pyscenic 分析**：包括 GRN 构建、TF-motif 富集分析和 AUCell 分析，生成最终的 regulon 活性评分文件。
4. 4.**可视化**：在 R 中加载分析结果，进行 RSS 分析和热图绘制，展示转录因子的细胞类型特异性和活性。

此流程旨在帮助研究人员深入理解单细胞数据中转录因子的调控网络及其在特定细胞类型中的活性。
